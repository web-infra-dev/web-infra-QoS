```ts title="rspress.config.ts"
import { defineConfig } from 'rspress/config';

export default defineConfig({
  markdown: {
    // 使用 JS 版本的 MDX 编译器
    mdxRs: false,
  },
});
```

:::tip 注意
`mdxRs` 能力底层基于 Rspress 自研的 [@rspress/mdx-rs](https://github.com/web-infra-dev/mdx-rs) 库来实现，性能比 JS 版本的 MDX 编译器提升 10 倍以上，但不支持 JS 的插件。
:::
